# microservices-mini

# Project setup
Create a react-app using command:  npx create-react-app client

Create new folder --> posts
	mkdir posts
	cd posts
	npm init -y (generate package.json file)
	npm install express cors axios nodemon
	
Create new folder --> comments:
	mkdir comments
	cd comments
	npm init -y (generate package.json file)
	npm install express cors axios nodemon
